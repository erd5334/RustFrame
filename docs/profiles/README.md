# Profiles

Profiles are JSON overrides applied to settings when selected in the UI.

## Location
Profiles are stored under the RustFrame config directory:
- Windows: %APPDATA%\RustFrame\Profiles\windows
- macOS: ~/Library/Application Support/RustFrame/Profiles/macos
- Linux: ~/.config/RustFrame/Profiles/linux

Built-in profiles are seeded from resources/profiles/<os> on first run.

## Parameters
- Windows parameters are documented in windows.md.
- macOS and Linux parameters are not documented yet.
